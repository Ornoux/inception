---- DOCKER NOTIONS ----

- docker ps --> Liste des conteneurs existants
- docker run + <image name> --> Execute l'image
- docker image ls --> Liste les images existantes
- docker stop <container> --> Stoppe le conteneur
- docker rm <conteneur> --> Supprime le container

// DOCKER COMPOSE

- docker -v ---> Donne la version de Docker que l'on utilise afin de savoir quelle verison de Docker Compose utiliser.
